# Bug 001 – Changing theme redirects user to Home screen

---

## 📌 Summary
After changing the theme, the user is redirected to the Home screen instead of staying on the "My Account" page.

---

## 🧪 Environment
- Application: IMDb (Mobile)
- Version: 9.2.4.04
- Device: Realme RMX3085
- Platform: Android
- OS: Android 13

---

## ⚙️ Preconditions
- App is installed and running  
- User is logged in  

---

## 🔁 Steps to Reproduce
1. Open the IMDb app
2. Navigate to "My Account"
3. Tap on the "Display" section
4. Change the theme from Dark to Light
5. Observe the screen after applying the change

---

## ❌ Actual Result
User is redirected to the Home screen after changing the theme.

---

## ✅ Expected Result
Theme should be applied immediately, and the user should remain on the "My Account" screen.

---

## ⚠️ Severity
Medium

---

## 🚨 Priority
Medium
