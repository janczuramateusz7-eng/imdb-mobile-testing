# Bug 002 – Language screen does not follow selected app theme

---

## 📌 Summary
When Dark Mode is enabled, the language selection screen is displayed in Light Mode instead of Dark Mode.

---

## 🧪 Environment
- Application: IMDb (Mobile)
- Version: 9.2.4.04
- Device: Realme RMX3085
- Platform: Android
- OS: Android 13

---

## ⚙️ Preconditions
- App is installed and running  
- Dark Mode is enabled  

---

## 🔁 Steps to Reproduce
1. Open the IMDb app
2. Navigate to "My Account"
3. Tap on the "Display" section
4. Tap on the "App language" section
5. Observe the screen

---

## ❌ Actual Result
Language selection screen is displayed in Light Mode despite Dark Mode being enabled.

---

## ✅ Expected Result
Language selection screen should follow the currently selected app theme and be displayed in Dark Mode.

---

## ⚠️ Severity
Medium

---

## 🚨 Priority
Medium
