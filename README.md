# IMDb Mobile Testing Project

---

## 📌 Project Overview

This project contains manual testing of the IMDb native mobile application.

Testing was performed using a checklist approach and focused on core mobile functionality and usability.

---

## 🎯 Scope

- mobile application testing (Android)
- exploratory testing
- checklist-based testing

---

## 🧪 Tested Areas

- authentication and account
- search functionality
- movie details
- navigation
- UI and usability

---

## 🛠 Tools Used

- mobile device / emulator
- screen recorder
- Jira (bug tracking – documented)

---

## 📂 Repository Structure

- `/docs` → checklist and summary  
- `/bugs` → bug reports  
- `/jira` → story description  
- `/screenshots` → evidence  

---

## 🐞 Results

- 2 bugs identified and documented  
- issues related to UI / functionality / usability
