# Testing Checklist – IMDb Mobile App

---

## 📌 Overview

This checklist was used to validate functionality, performance, usability, compatibility, and recoverability of the IMDb mobile application.

---

## 🧪 Functional Testing

- verify application behavior on start and background mode  
- verify integration with third-party services (e.g., authentication, payments)  
- verify page scrolling functionality  
- verify navigation between modules  
- verify text truncation issues  
- verify proper error messages (e.g., network error)  
- verify onboarding/tutorials for new users  
- verify behavior under different network conditions (3G, 4G, Wi-Fi, no internet)  
- verify screen orientation (portrait / landscape)  

---

## ⚡ Performance Testing

- verify battery consumption under normal usage  
- verify performance when switching network (Wi-Fi ↔ mobile data)  
- verify CPU usage optimization  
- verify memory usage and resource handling (GPS, camera)  
- verify application stability over time  
- verify behavior during intermittent connectivity  

---

## 👤 Usability Testing

- verify button size and accessibility  
- verify consistency of button colors  
- verify keyboard behavior  
- verify undo/back actions  
- verify clarity of navigation menus  
- verify readability of text content  
- verify user prompts for large data downloads  
- verify language and localization support  
- verify synchronization of user actions  
- verify availability of user guidance/help  

---

## 📱 Compatibility Testing

- verify UI across different screen sizes  
- verify text readability  
- verify behavior during phone calls / interruptions  

---

## 🔄 Recoverability Testing

- verify application recovery after crash  
- verify behavior during power loss  
- verify reconnection after network interruption
